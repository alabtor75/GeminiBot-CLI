##  Fonctionnalités
- Posez des questions directement à Gemini AI via l'interface CLI.
- Historique des questions/réponses par jour.
- Cryptage sécurisé de votre clé API.
- Stockage des fichiers localement dans le dossier d'installation.
- Compatible Windows.
- Installation rapide avec un setup EXE professionnel.

---

##  Installation

1. **Téléchargez** l'installateur `GeminiBotInstaller.exe`.
2. **Lancez** le setup, acceptez la licence.
3. **Entrez votre clé API Gemini** ou **créez-en une nouvelle**.
4. **Terminé !** Utilisez GeminiBot CLI via le raccourci bureau ou le menu Démarrer.

---

##  Obtention de votre clé API Gemini

Si vous n'avez pas encore de clé API, créez-en une gratuitement ici :  
 [Créer ma clé API Gemini](https://aistudio.google.com/app/apikey)
